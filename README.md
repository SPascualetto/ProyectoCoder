1. Página de inicio. En la misma se puede encontrar los posteos realizados.
2. Registro de usuario.
3. Login, a partir del cual se puede realizar 
a. Agregar Posteo 
b. Modificar Posteo
c. Elimiar Posteo 
d. Editar Perfil
e. Acerca de mi 
f. Salir
